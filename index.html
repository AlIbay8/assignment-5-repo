How to Apply CSS to HTML

An external CSS file can be applied to an HTML file by including the following code within the head tags of the HTML file: 

            
            <link rel="stylesheet" href="styles.css" />
where the "href" attribute of the link tag has to be the path to the css file, relative to the html file.
