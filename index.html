How to Apply CSS to HTML

An external CSS file can be applied to an HTML file by including the following code within the head tags of the HTML file: 

<link rel="stylesheet" href="styles.css" />
where the "href" attribute of the link tag has to be the path to the css file, relative to the html file.

How to Use JavaScript in a Webpage

An external JavaScript file can be applied by adding the following element to the HTML file: 

<script src="path/to/my/script.js"></script>
where the "src" attrubute is the path to the JavaScript file, relative to the html file.
Alternatively, JavaScript can be written within the HTML file by writing JavaScript code within script tags. For example: 

<script>
    console.log("This is JavaScript running from the HTML file!")
</script>
